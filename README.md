# PDF Table Parser

Проект для извлечения таблиц из PDF-документов и конвертации их в Markdown формат.

## Структура проекта

```
PythonProject1/
├── hack/                    # Основной проект парсера таблиц
│   ├── main.py             # Точка входа для обработки PDF
│   ├── table_detector/     # Модуль детекции таблиц
│   ├── formatter/          # Модуль конвертации в Markdown
│   ├── pdf_files/          # Входные PDF файлы
│   └── output/             # Выходные Markdown файлы
└── 123/                    # Дополнительный проект парсера
    └── pdf_table_parser/   # Альтернативная реализация
```

## Основные возможности

- **Детекция таблиц с рамками**: Использует pdfplumber для поиска таблиц с явными границами
- **Детекция безрамочных таблиц**: Использует Table Transformer (microsoft/table-transformer-detection) для ML-детекции
- **Автоматическое слияние таблиц**: Объединение таблиц, разбитых между страницами
- **Фильтрация качества**: Отсеивание сканов, шумных таблиц и водяных знаков
- **Конвертация в Markdown**: Форматирование таблиц в стандартный Markdown

## Установка

См. `hack/README.md` для подробных инструкций по установке и использованию.

## Быстрый старт

```bash
cd hack
python main.py
```

Это обработает все PDF файлы из директории `pdf_files/` и сохранит результаты в `output/`.
